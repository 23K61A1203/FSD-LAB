Default Information:
JavaScript provides different ways to display output:
  • alert() → Displays a popup alert box.
  • document.write() → Writes directly to the webpage (not recommended after page load).
  • innerHTML → Updates content of an HTML element dynamically.
  • console.log() → Sends output to the browser console (used for debugging).
These methods help programmers show results, debug code, or interact with users effectively.
