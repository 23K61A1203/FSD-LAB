3. Using HTML Form Input

Default Information:
JavaScript provides multiple ways to take input from the user:
  • prompt() → To directly ask the user for a text value.
  • confirm() → To get a Yes/No (true/false) response.
  • HTML Forms → To collect structured input like text, numbers, dates, etc.
These methods help developers interact with users and collect data for processing.
