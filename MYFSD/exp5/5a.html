Different Ways to Use Colors in CSS

Named Colors: CSS supports 140+ predefined color names like red, blue, green. These are easy to remember and quick to apply.
HEX Codes: Colors are defined using hexadecimal values, e.g. #ff5733. This is commonly used in web design because it is precise.
RGB: Defined by mixing red, green, and blue values (0â€“255). Example: rgb(0, 128, 255) gives a bright blue.
RGBA: Same as RGB but with an Alpha channel for transparency. Example: rgba(0,128,255,0.6) makes the box slightly see-through.
HSL: Defined by Hue (color angle), Saturation (%), and Lightness (%). Example: hsl(200, 100%, 50%) gives vivid blue.
HSLA: Same as HSL but with transparency support. Example: hsla(200, 100%, 50%, 0.5) creates a semi-transparent blue.
