css font and text demonstration

This example demonstrates how different CSS font and text properties can completely change the appearance of your webpage. With just a few styles, text can become more beautiful, readable, and attractive.

Font Size: Controls how big or small the text appears.
Font Weight: Makes the text light, normal, or bold.
Font Style: Allows italic or normal styling.
Text Decoration: Adds underline, overline, or strike-through.
Text Transform: Changes text to uppercase, lowercase, or capitalize.
Text Alignment: Aligns text to left, right, center, or justify.

Notice how this highlighted text looks different compared to normal text because of applied CSS properties.
