Understanding Hyperlinks in HTML

1. The <a> Tag

The <a> tag is used to create hyperlinks in HTML. It is the anchor tag that wraps the clickable text or content.

Example:

<a href="https://www.example.com">Click Here</a>

Output:

Click Here

2. The href Attribute

The href attribute specifies the URL or link destination where the user will be taken when the link is clicked.

Example:

<a href="https://www.openai.com">OpenAI</a>

Output:

OpenAI

3. The target Attribute

The target attribute specifies where to open the linked document. Common values include:
  • _self (default) - Opens the link in the same window/tab.
  • _blank - Opens the link in a new window or tab.
  • _parent - Opens the link in the parent frame.
  • _top - Opens the link in the full body of the window.

Example without target (opens in same tab):

<a href="https://www.wikipedia.org">Wikipedia</a>

Output:

Wikipedia (Same Tab)

Example with target="_blank" (opens in new tab):

<a href="https://www.wikipedia.org" target="_blank">Wikipedia</a>

Output:

Wikipedia (New Tab)
